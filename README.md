# TestEdusol

Dieses Projekt enthält drei Versionen der Edusol-Website zum Vergleich.

## Einstieg

Öffne **[[index.html](index.html)](https://b4sti12.github.io/TestEdusol/)** im Browser — dort kannst du direkt auswählen, welche Version du ansehen möchtest.

## Ordner-Struktur

- **[Original/](Original/)** – Die ursprüngliche Version der Website
- **[Humanized/](Humanized/)** – Die überarbeitete/humanisierte Version
- **[V2/](V2/)** – Kompletter Neuaufbau mit Poppins-Font, Bootstrap 5, Dropdown-Navigation und erweitertem Seitenumfang
